Hardware

Power-On

The uCritAir ships in a protected power-off mode, to prevent draining the battery before it arrives. To turn it on for the first time, simply press the reset button with a small object such as a paperclip, and wait a moment.

Displays

E-Ink

The uCritAir's topmost display is a non-interacive e-ink display which is used to communicate device status, air quality, gameplay metrics, and more. The e-ink display can be manually updated from the system menu.

When the device is flipped upside-down, the e-ink display will flip its orientation to match.

LCD

The main display is a touchscreen LCD which is constantly updated in real time. It is where the action happens. All air quality and gameplay features are controlled through interfaces that appear on the LCD.

Face Buttons

The D-pad, A/B buttons, and start/select buttons are used to control the uCritAir's software in a manner that will be familiar to anyone who has used a gamepad. That is, in general: D-pad is for navigation, A/B are for confirm and cancel respectively, start is for entering or exiting some overarching context, and select is for changing modes within a context.

USB-C Port

Connecting the μCritAir to a power source via the USB-C port will refill its battery.

New builds of the system software are flashed to the μCritAir over the USB-C port.
